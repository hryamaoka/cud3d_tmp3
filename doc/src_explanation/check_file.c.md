# `src/parser/check_file.c` 詳細解説

## 概要
コマンドライン引数として渡されたファイル名が有効な形式（`.cub`拡張子）であるかを検証します。

## コード詳細解説

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **15** | `int check_filename(char *filename)` | **ファイル名検証関数**<br>引数で渡されたパスが有効な拡張子を持つか確認します。 |
| **19-20** | `  if (!filename) return (0);` | ポインタがNULLの場合はエラー(`0`)を返します。 |
| **21** | `  extention = ft_strrchr(filename, '.');` | **拡張子検索**<br>文字列の最後にあるドット`.`を検索し、その位置へのポインタを取得します。 |
| **22** | `  if (!extention)` | ドットが見つからない（拡張子がない）場合はエラー。 |
| **24** | `  diff = ft_strncmp(extention, ".cub", 5);` | **拡張子比較**<br>取得した拡張子部分と文字列".cub"を比較します。長さ5（4文字+NULL）で比較し、完全一致を確認します。 |
| **25** | `  if (diff)` | 一致しない（差分がある）場合はエラー。 |
| **27** | `  return (1);` | 検証を通過した場合、有効(`1`)を返します。 |
