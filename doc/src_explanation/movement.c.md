# `src/input/movement.c` 詳細解説

## 概要
壁との衝突判定（コリジョン）を考慮したプレイヤーの移動処理を実装します。

## コード詳細解説

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **16** | `void move_forward(t_game *game)` | **前進**<br>向いている方向へ移動します。 |
| **24-31** | `  if (dir > 0) check_x = ...` | **衝突用オフセット**<br>壁のギリギリまで近づけるよう、少し先(`COLLISION_BUFFER`)の地点で壁判定を行います。 |
| **32-35** | `  if (new_x ... && ... != '1')` | **X軸移動**<br>移動先が壁でも閉じたドアでもなければ座標を更新します。 |
| **36-39** | `  if (new_y ...)` | **Y軸移動**<br>XとYを独立して判定することで、壁沿いのスムーズな移動（スライド）を実現します。 |
| **42/68/98** | `void move_back/left/right` | 後退および蟹歩き（左右平行移動）の実装です。方向ベクトル計算が異なるのみで、衝突判定ロジックは共通です。 |
