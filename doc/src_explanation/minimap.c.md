# `src/render/minimap.c` 詳細解説

## 概要
ゲーム画面上に現在のマップ状況とプレイヤー位置を表示するミニマップ機能です。デバッグや探索の補助として機能します。

## コード詳細解説

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **15** | `static void draw_rect(...)` | **矩形描画**<br>指定された座標に、指定サイズの正方形を指定色で描画します。ミニマップの各セル（壁や床）の描画に使用します。 |
| **24-28** | `      if (x + j < 0 ...)` | **境界チェック**<br>画面外への描画を防ぎ、セグメンテーションフォールトを回避します。 |
| **42** | `void draw_minimap(t_game *game)` | **ミニマップ描画メイン**<br>マップデータ全体を走査し、壁と床を描画した後、プレイヤー位置を描画します。 |
| **52-53** | `      draw_x = MM_OFFSET_X + ...` | マップ座標を画面上のピクセル座標に変換します。`MM_SCALE`倍してオフセットを加算します。 |
| **61-62** | `        if (... == '1') ...` | 壁('1')なら壁の色で描画。 |
| **63-64** | `        else if (... != ' ') ...` | 空白以外（床、アイテム、プレイヤースポーン地点）なら床の色で描画。 |
| **72-73** | `  int px = ...; int py = ...;` | プレイヤー座標(`double`)をピクセル座標(`int`)に変換します。 |
| **75** | `  draw_rect(..., MM_COLOR_PLAYER);` | プレイヤーを赤色等で描画します。 |
