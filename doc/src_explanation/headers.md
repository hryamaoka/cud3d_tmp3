# 内部ヘッダーファイル 詳細解説

## 概要
各機能モジュール（Core, Engine, Input, Parser, Render, Utils）内でのみ使用、あるいはモジュール間のインターフェース定義として機能するヘッダーファイル群です。

## src/core/init.h

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **18** | `void init_parse(t_game *game);` | パース構造体の初期化関数のプロトタイプ。 |
| **19** | `void init_player_dir(t_game *game);` | プレイヤー初期方向設定関数のプロトタイプ（`parse_player.c`で使用）。 |

## src/engine/raycast.h

| 行番号 | コード | 解説 |

| **19** | `void select_texture(...);` | テクスチャ選択ロジックの宣言。 |
| **20-21** | `void init_ray_step_x/y(...);` | レイの初期ステップ計算関数の宣言。 |

## src/input/input.h

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **18-20** | `ROT_SPEED`, `MOVE_SPEED`, ... | **定数定義**<br>ゲームプレイの調整パラメータ定義です。<br>`ROT_SPEED 0.05`: キー操作時の回転速度。<br>`MOVE_SPEED 0.1`: 移動速度。<br>`COLLISION_BUFFER 0.2`: 壁との最小接触距離。 |
| **22-29** | `void move_forward(...)` ... | 移動、回転、ドア操作を行う各種アクション関数のプロトタイプ宣言。 |

## src/input/keys.h

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **4-11** | `KEY_ESC`, `KEY_W`, ... | **キーコード定義**<br>X11環境（Linux）でのキーコードマッピングです（注: macOSでは異なる場合があります）。<br>`65307`: ESCキー<br>`119`: 'w'キー |
