# `src/core/game_init.c` 詳細解説

## 概要
ゲーム構造体 `t_game` の初期化処理を行います。パース前のデフォルト値の設定や、ゼロクリアを担当します。

## コード詳細解説

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **16** | `void init_parse(t_game *game)` | **パース用初期化**<br>設定ファイル読み込み前に、各値を「未設定」状態を表す値（NULLや-1）に設定します。これにより、パース後の重複定義チェックや欠落チェックが可能になります。 |
| **21-26** | `  game->floor.r = -1; ...` | 色の初期値を-1に設定（0は有効な黒色であるため）。 |
| **37** | `void init_game(t_game *game)` | **ゲーム全体初期化**<br>アプリケーション起動直後に呼ばれ、全メモリ領域を安全な状態にします。 |
| **40** | `  ft_bzero(game, sizeof(t_game));` | **ゼロクリア**<br>構造体全体を0で埋めます。これにより、未初期化のポインタがゴミ値を指す事故を防ぎます。 |
| **41-42** | `  game->z_buffer = NULL; ...` | 明示的にNULLを設定（bzeroで既に0ですが、可読性のため）。 |
| **46** | `    game->wall_tex[i].img = NULL;` | テクスチャ画像ポインタもNULL初期化。画像ロード判定に使用されます。 |
| **53** | `  init_parse(game);` | パース変数の具体的初期化を行います。 |
