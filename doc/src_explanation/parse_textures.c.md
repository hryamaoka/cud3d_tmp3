# `src/parser/parse_textures.c` 詳細解説

## 概要
設定行からテクスチャファイルのパス（NO, SO, WE, EA）を抽出し、拡張子の正当性などを検証して構造体に保存します。

## コード詳細解説

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **17** | `static int set_texture(...)` | **パス設定・検証**<br>抽出されたパス文字列を検証して保存します。 |
| **20** | `  if (*texture) return (0);` | **重複チェック**<br>既にその方向のテクスチャが設定済みであればエラーです。 |
| **22-27** | `  while (len > 0 ...)` | **トリミング**<br>パス末尾の改行や空白を除去します。 |
| **28** | `  if (len < 4 || ft_strncmp(...))` | **拡張子チェック**<br>末尾が`.xpm`で終わっているかを確認します。 |
| **30** | `  *texture = ft_strdup(path);` | パス文字列を複製して保存します。 |
| **34** | `int parse_texture(char *line, t_game *game)` | **テクスチャ行パース**<br>行をスペースで分割し、識別子に対応する方向へパスを設定します。 |
| **38** | `  split = ft_split(line, ' ');` | スペース区切りでトークン化します。 |
| **44-67** | `  if (!ft_strncmp("NO", ...))` | 第一トークン（識別子）を判定し、対応する構造体メンバへ`set_texture`を呼び出します。失敗時は即座にエラー出力して終了します。 |
