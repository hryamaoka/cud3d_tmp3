# `src/parser/parse_player.c` 詳細解説

## 概要
マップ読み込み中に検出されたプレイヤーの開始位置と向きを処理します。

## コード詳細解説

| 行番号 | コード | 解説 |
| :--- | :--- | :--- |
| **15** | `void set_direction(t_game *game)` | **ベクトル設定**<br>文字（N,S,W,E）に応じて、方向ベクトル(`dir`)とカメラ平面ベクトル(`plane`)を設定します。FOV（視野角）は`dir`と`plane`の比率で決まります（ここでは約66度）。 |
| **40** | `static void set_player_info(...)` | 座標保存と方向設定を呼び出します。座標に`0.5`を足すことで、グリッドの中心に配置します。 |
| **47** | `int find_player(t_game *game)` | **プレイヤー探索**<br>マップ全域を走査してプレイヤー位置を特定します。 |
| **57** | `      if (ft_strchr("NSWE", ...))` | そのマスの文字がプレイヤー文字か判定。 |
| **65** | `  if (count != 1)` | **人数チェック**<br>マップ全体でプレイヤーが「ただ1人」でなければエラーとします（0人や2人以上は不可）。 |
