# `src/utils/utils.c` 詳細解説

## 概要
プロジェクト全体で使用する汎用的な計算関数などを定義します。

## 依存関係
*   `cub3d.h`: 標準的な定義。

## 関数詳細: ft_abs

### プロトタイプ
`int ft_abs(int n)`

### 戻り値
引数 `n` の絶対値（非負整数）。

### 解説
整数の絶対値を返します。`math.h` の `fabs`（浮動小数点用）とは異なり、こちらは整数 (`int`) 用です。テクスチャの高さ計算や描画範囲計算などで、負の値を正に補正するために使用されます。

| 行番号 | コード | 詳細解説 |
| :--- | :--- | :--- |
| 16-17 | `if (n < 0)` | 負の数であれば、符号を反転させて返します。 |
| 18 | `return (n);` | 正の数（または0）であれば、そのまま返します。 |
