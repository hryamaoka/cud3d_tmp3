# `src/input/keys.h` 詳細解説

## 概要
キーボード入力のキーコード（X11/Linux標準）をマクロ定義します。環境（OS）によって変更が必要になる場合がありますが、現在はLinux/WSL向けの定義となっています。

## 定数定義

| マクロ名 | 値 | 対応キー |
| :--- | :--- | :--- |
| `KEY_ESC` | 65307 | Escape キー |
| `KEY_W` | 119 | W キー |
| `KEY_A` | 97 | A キー |
| `KEY_S` | 115 | S キー |
| `KEY_D` | 100 | D キー |
| `KEY_E` | 101 | E キー（ドア操作等） |
| `KEY_LEFT` | 65361 | 左矢印キー |
| `KEY_RIGHT` | 65363 | 右矢印キー |
