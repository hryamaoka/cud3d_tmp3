# `src/parser/parse_utils.c` 詳細解説

## 概要
パース処理中に頻繁に使用される、汎用的な文字列判定ユーティリティ関数を提供します。実質的には `libft` の拡張のような役割を果たします。

## 依存関係
*   `parse.h`: パース用ヘッダ。

## 関数詳細: is_empty_line

### プロトタイプ
`int is_empty_line(char *line)`

### 戻り値
*   `1`: 空行である（空白文字のみで構成されている、または長さ0）。
*   `0`: 空行ではない（少なくとも1つの可視文字が含まれる）。

### コード行別解説

| 行番号 | コード | 詳細解説 |
| :--- | :--- | :--- |
| 15 | `int is_empty_line(char *line)` | **空行判定関数**<br>引数の文字列に、意味のある文字が含まれているかを確認します。 |
| 19-22 | `while (line[i]) ...` | 文字列を先頭からスキャンします。 |
| 20 | `if (line[i] != ' ' ...)` | スペース、タブ、改行以外の文字（英数字記号など）が見つかった瞬間、それは「空行ではない」と判断できます。 |
| 21 | `return (0);` | 即座に `0` を返します。 |
| 24 | `return (1);` | 文字列の最後まで到達した場合（全ての文字が空白文字だった場合）、`1` を返します。 |
