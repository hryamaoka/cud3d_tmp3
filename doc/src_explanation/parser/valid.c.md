# `src/parser/valid.c` 詳細解説

## 概要
読み込まれた色設定値が有効な範囲内にあるかを最終確認します。パース時にもチェック等は行われますが、ゲーム開始直前の安全確認として機能します。

## 依存関係
*   `parse.h`: `t_game`, `t_color` 構造体。

## 関数詳細: valid_color

### プロトタイプ
`int valid_color(t_game *game)`

### 戻り値
*   `1`: 有効（全ての色成分が適正範囲内）。
*   `0`: 無効（範囲外の色成分が存在）。

### コード行別解説

| 行番号 | コード | 詳細解説 |
| :--- | :--- | :--- |
| 16-21 | `if (game->ceiling.r <= 0 ...)` | **天井色チェック**<br>R, G, B の各成分が `0` 以下または `255` 以上であるかを判定します（※コード上は `0` と `255` も無効として扱われますが、初期値 `-1` を弾く意図が含まれます）。 |
| 22-27 | `if (game->floor.r <= 0 ...)` | **床色チェック**<br>天井と同様に床の色もチェックします。 |
| 28 | `return (1);` | 全ての問題がなければ `1` を返します。 |
