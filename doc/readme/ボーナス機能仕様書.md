# 実装したボーナス機能

## 衝突判定 (Wall Collision)
- プレイヤーが壁 (`1`) を通り抜けられないように制御しています。
- 移動先の座標が壁である場合、移動をキャンセルします。
- 壁との距離に少し余裕（バッファ）を持たせて、壁にめり込まないようにしています (Slide/Glide 挙動)。

## ミニマップシステム (Minimap)
- 画面左上に、現在のマップとプレイヤーの位置を表示するミニマップを実装しました。
- プレイヤーの移動に合わせてリアルタイムに更新されます。
- 壁は白、床は黒、プレイヤーは赤で表示されます（色は実装に準拠）。


## マウスによる視点回転
マウスを左右に動かすと視点が回転します。

## 開閉するドア
- マップ内の `D` をドアとして認識します。
- ドアの前で `E` キーを押すと、開閉（`D` ↔ `O`）できます。

## アニメーションスプライト
- マップ内の `2` をスプライトとして表示します。
- 緑と赤の球体が点滅（アニメーション）します。
